# Pathfinding Visualizer (Java)

An interactive Pathfinding Visualizer built using Java Swing that demonstrates how BFS, DFS, and A* algorithms solve mazes in real time.

## Features

* Maze Generation
* BFS (Breadth First Search)
* DFS (Depth First Search)
* A* Pathfinding Algorithm
* Real-Time Visualization
* Speed Control
* Statistics Dashboard
* Interactive Wall Drawing
* Custom Start and End Nodes
* Dark Theme UI

## Technologies Used

* Java
* Java Swing
* Object-Oriented Programming (OOP)
* Data Structures & Algorithms

## Algorithms Implemented

### BFS

Finds the shortest path in an unweighted maze.

### DFS

Explores one path deeply before backtracking.

### A*

Uses a heuristic to efficiently find the shortest path.
